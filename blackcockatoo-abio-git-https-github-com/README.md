# The Moss Man / B$S — Moss Tree

A zero-build, GitHub Pages-ready progressive web app for The Moss Man and Blue $nake Studio.

The interface is a nine-node angular project tree influenced by calm console navigation. It supports touch swipe, mouse wheel, arrow keys, direct selection, URL hashes, reduced motion and a print-friendly expanded résumé. Frankston → Fuji has its own bilingual sister-city song branch, Black Omen links to its Bunurong–Boonwurrung research map, and the complete OSS-734G Semantic Sovereignty field manual is included as a linked companion page.

## Run as a web app

The Moss Tree is an installable progressive web app. Serve the repository root over HTTPS or localhost so the manifest, install prompt and offline worker are available. Opening `index.html` directly still works for browsing, but browsers disable installation and service workers on `file://` URLs.

For a quick local preview, run any static server from the repository root, then open its localhost URL. Once loaded online, the portfolio shell and field manual are available offline. Deployment remains GitHub Pages and Netlify compatible with no build step.
