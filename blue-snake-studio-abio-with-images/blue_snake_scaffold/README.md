# The Moss Man / Blue $nake Studio scaffold

Merged scaffold for **The Moss Man / Blue $nake Studio / B$S**.

This version folds the useful content from the `Blackcockatoo/abio` repo into the newer React/Vite scaffold:

- Meta-Pet privacy-first digital companion
- Moss 60 visual mathematics / digital DNA system
- Auralia / B$S mythic visual identity
- Black Wing Crew / Neon Venom music world
- Teacher’s Secret Cheatsheet / behaviour support resources
- The Grammarian’s Coup Memetic Lexicon
- Contact links and live project links

## Run locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Main edit points

- `client/src/pages/Home.tsx` — merged bio website content
- `client/src/index.css` — B$S visual styling and custom classes
- `client/index.html` — title and SEO description
